MULTIMEDIA TAG

  • img : Used to embed images in a document. It is a self-closing tag with a mandatory src attribute for the image URL and an alt attribute for alternative text, which is important for accessibility.
  • audio : Used to embed sound content, such as music or voice recordings. It can use a src attribute or contain source elements for multiple file formats to ensure cross-browser compatibility.
  • video:Used to embed video content and its associated playback controls. Like audio, it can use a src attribute or multiple source elements. It supports additional attributes like width, height, and poster (an image to display before the video plays).
  • iframe : Used to embed another HTML document (or external content like a YouTube video) within the current page.
